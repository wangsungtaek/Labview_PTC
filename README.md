# PTC(Power Temp Cycle)
### 개요
> 반도체 소자의 발열/방열 사이클과 외부 온도 사이클을 혼합하여,<br>
> 실제적인 환경변화가 있는(예: 차량용 반도체) 반도체의 내구성을 시험하는 소프트웨어<br>

### 주요기능
1. 챔버 통신제어(TCP/IP)
2. 120ch 파워 통신제어
3. DAQ 1M/Sample 데이터 수집
